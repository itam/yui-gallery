gallery-sm-dragdrop Module

Defined in: src/sm-dragdrop/js/dragdrop.js:19

Provides the DragDrop class, a simpler and more efficient drag and drop implementation than the one in YUI core.

This module provides the following classes:

  • DragDrop
  • Plugin.DragDrop.Reorder

This module is a rollup of the following modules:

  • gallery-sm-dragdrop-reorder
    Provides the Plugin.DragDrop.Reorder class, a DragDrop plugin that adds drag-to-reorder functionality to a collection of nodes.

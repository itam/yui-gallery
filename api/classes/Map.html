_is

(
  • a
  • b
)
Boolean protected

Defined in src/sm-map/js/map.js:282

Returns true if the two given values are the same value, false otherwise.

This is an implementation of the ES5/6 SameValue comparison algorithm, which is more correct than === in that it considers NaN to be the same as NaN, and does not consider 0 to be the same as -0.

This method is an alias for Object.is() if available, or a fallback implementation if not.

Parameters:

  • a Mixed

    First value to compare.

  • b Mixed

    Second value to compare.

Returns:

Boolean: true if a and b are the same value, false otherwise.

Sets the value of the entry with the given key. If the key already exists, its value will be overwritten; otherwise it will be created.

The key may be any JavaScript value (including both primitives and objects), but string keys will allow fast lookups, whereas non-string keys will result in slower lookups.

Parameters:

  • key Mixed

    Key to set.

  • value Mixed

    Value to set.
